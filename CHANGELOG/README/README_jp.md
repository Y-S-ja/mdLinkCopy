# Quick Md Copy

> **一瞬で Markdown リンクをコピー。もっとスマートに、もっとクリエイティブに。**

`Quick Md Copy` は、Web ページのタイトルと URL を Markdown 形式 `[タイトル](URL)` で瞬時に取得するための Chrome 拡張機能です。
単なる URL コピーにとどまらず、**「特定の選択範囲へ直接ジャンプできるリンク（Scroll to Text）」** の自動生成や、Markdown 構文と干渉する記号（`[` `]` `|`）の柔軟な処理など、ドキュメント作成を加速させる機能を備えています。

---

## 🌟 主な機能

### 📍 高機能なテキストリンク（Scroll to Text）生成

テキストを選択してコピーするだけで、その場所まで自動的にスクロールするリンクを生成します。
長い文章でも、設定した「しきい値」を超えると「`文頭`,`文末`」の形式でコンパクトにリンク化。文脈を壊さずに引用できます。

### 🎨 自由自在なカスタマイズ

- **コピー成功/失敗の通知**: お好みのメッセージに書き換え可能。
- **タイトル記号の自動処理**: `[` や `]`、`|` などの記号を「エスケープ」「全角化」「削除」「そのまま」から選択して処理。
- **読みやすい URL**: URL エンコードされた読みにくい日本語 URL も、デコードして美しくコピー。

### ⌨️ パワフルなショートカット

マウスを使わずに作業を完結させたい方のためのショートカット：

- ページへのリンクをコピー
- 選択箇所へのテキストリンクをコピー

> [!NOTE]
> ショートカットキーはオプション画面で変更可能です。

---

## 🚀 使い方

### 方法① アイコンをクリック

ツールバーの `Quick Md Copy` アイコンをクリックすると、現在開いているページのタイトルと URL をコピーします。

### 方法② 右クリックメニューから

Web ページ上のテキストを選択して右クリックし、「選択箇所への Markdown リンクをコピー」を選択します。選択範囲へのジャンプ機能付きリンクが生成されます。

### 方法③ ショートカットキー

ショートカットキーを押すだけで、マウスを操作せずにコピーが完了します。

---

## 🔧 設定のヒント

オプション画面では、快適なコピーを実現するための詳細な設定が可能です。

- **しきい値**: テキストがこの長さを超えると、短縮リンク（`start`,`end` 形式）に切り替わります。
- **抽出するの長さ**: 短縮リンクの「文頭」と「文末」に使用する文字数を調整できます。
- **リセット機能**: 各設定項目にある 🔄 ボタンで、いつでも初期設定に戻せます。

## 🔒 プライバシーと安全性

- **オフライン動作**: 全ての処理はブラウザ内のローカル環境で完結します。
- **データ収集なし**: あなたの閲覧履歴や個人情報がサーバーに送信されることは一切ありません。
