> **Copy Markdown links in an instant. Work smarter, stay creative.**

`Quick Md Copy` is a Chrome extension that lets you instantly capture the title and URL of web pages in Markdown format: `[Title](URL)`.
Far more than a simple URL copier, it speeds up your documentation workflow with features like **Scroll to Text** deep links and flexible handling of characters that conflict with Markdown syntax (`[` `]` `|`).

## 🌟 Key Features

### 📍 Advanced Text Fragment (Scroll to Text) Support

Simply select text and copy to generate a link that automatically scrolls to that specific spot.
For long passages, the extension intelligently creates a compact "`start`,`end`" fragment if it exceeds your defined threshold, allowing you to quote precisely without cluttering your links.

### 🎨 Fully Customizable Experience

- **Success/Failure Notifications**: Customize the feedback message to suit your style.
- **Smart Symbol Handling**: Choose how to handle `[`, `]`, and `|` characters in titles—options include `Escape`, `Full-width`, `Remove`, or `Keep as is`.
- **Readable URLs**: Automatically decodes encoded Japanese or special character URLs into a clean, human-readable format for your documents.

### ⌨️ Powerful Shortcuts

Shortcuts for those who want to get things done without touching the mouse:

- Copy page-wide Markdown link
- Copy deep link for the current selection

> [!NOTE]
> Keyboard shortcuts can be customized in the browser's extension management settings.<br>
> (Right-click icon > Manage Extension > Keyboard shortcuts)

## 🚀 How to Use

### Method 1: Click the Icon

Click the `Quick Md Copy` icon in your toolbar to instantly copy the title and URL of the active tab.

### Method 2: From the Context Menu

Select text on a web page, right-click, and choose "Copy Markdown link for selection." This generates a link with a jump-to-selection fragment.

### Method 3: Keyboard Shortcuts

Simply press your keyboard shortcuts to finish your copy without ever touching your mouse.

## 🔧 Settings Tips

The options page allows for fine-tuned control over your link generation:

- **Threshold**: When text length exceeds this limit, links switch to the compact "`start`,`end`" format.
- **Extraction Length**: Adjust the number of characters taken from the "start" and "end" of shortened links.
- **Reset to Default**: Use the 🔄 button next to any setting to instantly revert to the original configuration.

## 🔒 Privacy & Security

- **Offline Operation**: All processing happens locally within your browser.
- **No Data Collection**: Your browsing history and personal information are never sent to any server.
